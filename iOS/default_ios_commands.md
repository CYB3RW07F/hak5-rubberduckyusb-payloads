# Default iOS commands for Rubber Ducky 3.0 (Tested on iOS 26.4)

| Action                         | iOS Keys | DuckyScript 3.0                                                        |
| ------------------------------ | -------- | ---------------------------------------------------------------------- |
| Open Spotlight Search          | ⌘ Space  | COMMAND SPACE                                                          |
| Go to Home Screen              | ⌘ H      | COMMAND h                                                              |
| Open App Switcher (cycle apps) | ⌘ Tab    | COMMAND TAB                                                            |
| Screenshot                     | ⌘ ⇧ 3    | COMMAND SHIFT 3                                                        |
| Screenshot + Markup            | ⌘ ⇧ 4    | COMMAND SHIFT 4                                                        |
| Select All                     | ⌘ A      | COMMAND a                                                              |
| Copy                           | ⌘ C      | COMMAND c                                                              |
| Paste                          | ⌘ V      | COMMAND v                                                              |
| Cut                            | ⌘ X      | COMMAND x                                                              |
| Undo                           | ⌘ Z      | COMMAND z                                                              |
| Redo                           | ⌘ ⇧ Z    | COMMAND SHIFT z                                                        |
| Bold                           | ⌘ B      | COMMAND b                                                              |
| Italic                         | ⌘ I      | COMMAND i                                                              |
| Underline                      | ⌘ U      | COMMAND u                                                              |
| Find in Page                   | ⌘ F      | COMMAND f                                                              |
| Open URL Bar (Safari)          | ⌘ L      | COMMAND l                                                              |
| New Tab (Safari)               | ⌘ T      | COMMAND t                                                              |
| Close Tab (Safari)             | ⌘ W      | COMMAND w                                                              |
| Reload Page (Safari)           | ⌘ R      | COMMAND r                                                              |
| Next Tab (Safari)              | ⌘ ⇧      | COMMAND SHIFT ]                                                        |
| Prev Tab (Safari)              | ⌘ ⇧      | COMMAND SHIFT [                                                        |
| Move cursor word right         | ⌥ →       | ALT RIGHT                                                              |
| Move cursor word left          | ⌥ ←       | ALT LEFT                                                               |
| Select word right              | ⌥ ⇧ →     | ALT SHIFT RIGHT                                                        |
| Select to line start           | ⌘ ⇧ ←     | COMMAND SHIFT LEFT                                                     |
| Select to line end             | ⌘ ⇧ →     | COMMAND SHIFT RIGHT                                                    |
| Delete word left               | ⌥ ⌫      | ALT BACKSPACE                                                          |
| Siri (hold ⌘)                  | Hold ⌘   | HOLD COMMAND / DELAY 1500 / RELEASE COMMAND                            |

---
## Please note
The commands shown in the table above have been tested on an iPhone 16 Pro running iOS 26.4. This table is not guaranteed to work and is subject to changes after any iOS update and/or patches.
