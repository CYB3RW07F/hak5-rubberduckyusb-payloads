# Rubber Ducky Payloads 🦆

Welcome to this repository of **Rubber Ducky payloads** a curated collection of scripts designed to be used with USB HID injection devices like the Hak5 Rubber Ducky.

## 💡 What is this?

This repo contains ready-to-use payloads that simulate keyboard input to automate tasks, demonstrate security concepts and support ethical hacking and penetration testing workflows.

## ⚠️ Disclaimer

All payloads are provided for **educational and authorized testing purposes only**. Do not use these scripts on systems you do not own or have explicit permission to test. For more information, please consult the extended disclaimer [here](https://github.com/CYB3RW07F/hak5-rubberduckyusb-payloads/DISCLAIMER.md).

## 📂 What's inside?

* 🔐 Security-focused payloads (e.g., credential harvesting demos, lock screen triggers)
* ⚙️ Automation scripts for repetitive tasks
* 🧪 Proof-of-concept payloads for learning and experimentation
* 🖥️ Multi-platform support (Windows, Linux, macOS where applicable)

## 🚀 Getting Started

1. Copy a payload script
2. Modify it to fit your environment
3. Encode it using the Rubber Ducky encoder
4. Load it onto your device and test responsibly
